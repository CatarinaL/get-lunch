What's this?

This project was created by Kompal Joshi and Catarina Louren√ßo, as part of the CodeFirst:Girls Python Course that ran in DIT, 2018.

The main functionality of this project is finding you places for lunch, particularly on those days when you have no idea what you feel like, or no one in your group can make a decision.

It's powered by Yelp's API (Fusion), from where all the information about location, opening hours, type of food and other categories can be requested. It uses Python (Flask) for the backend and Javascript to handle the frontend.

And by the way, we take no sort of responsibility for how accurate those informations are, or more importantly, how bad your lunch was.
